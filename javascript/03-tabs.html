Abas

Um plugin em jQuery que provem, dinamicamente, a funcionalidade de transicionari através de painéis de conteúdo local. Faz uso de transições de CSS para criar a animação.

Vem com uma API de marcação que permite rodar o plugin sem precisar escrever nem uma linha sequer de JavaScript. Essa é a API de primeira classe do Responsive e deveria ser a sua primeira opção a ser considerada ao usar um plugin.

Acessibilidade

O suporte ao teclado vem com gatilhos padrões que respondem a tecla de espaço e ao enter. Também pode ser usado para navegar entre painéis pelo uso da tecla para a ESQUERDA, e para a DIREITA.

Suporte a escritas da direita para a esquerda

Adicionar o atributo dir="rtl" a tag html irá reverter todo o comportamento direcional da inserção de dados.

Por padrão o plugin de abas irá mostrar o primeiro painél. Para mudar o painél ativo padrão, adicione na marcarção o atributo aria-selected="true" ao alvo ("target") do elemento <a/>. 

<div class="tabs" data-tabs>
    <ul>
        <li><a href="#">aba1</a></li>
        <li><a href="#" aria-selected="true">aba2</a></li>
        <li><a href="#">aba3</a></li>
    </ul>

Métodos

O plugin de abas apresenta os seguintes métodos de assinatura.

.tabs(index)
Inicializa o plugin com um parâmentro opicional de index, (índice). Um indicador baseado em zero para definir qual aba mostrar.

Eventos

O plugin de abas apresenta os seguintes eventos para permitir que o desenvolvedor mexa em seu comportamento.

show.r.tabs
Esse evento é iniciado automaticamente assim que a instância do método show é invocada.
shown.r.tabs
Evento que é inicializado quando a instância do método shown é invocada, uma vez que a aba tenha completado sua animação e o novo painél está visível.

A conexão com esses é eventos é demonstrada abaixo:

// Conectando ao evento "show".
$("O_SEU_SELETOR").on("show.r.tabs", function(evento) {
    // Seu comportamento customizado...
});
                                               
// Conectando ao evento "shown".
$("O_SEU_SELETOR").on("shown.r.tabs", function(evento) {
    // Seu comportamento customizado...
});

API de dados

O comportamento padrão do plugin de abas é limitado pelo uso de uma API de marcação, que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Caso você precise desativar esse comportamento, o código abaixo irá lhe permitir atingir isso. 

// Desativar o carregamento do comportamento padrão
$(document).off(".r.tabs.data-api");
