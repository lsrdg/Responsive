Abas

Um plugin em jQuery que provem, dinamicamente, a funcionalidade de transicionari através de painéis de conteúdo local. Faz uso de transições de CSS para criar a animação.

Vem com uma API de marcação que permite rodar o plugin sem precisar escrever nem uma linha sequer de JavaScript. Essa é a API de primeira classe do Responsive e deveria ser a sua primeira opção a ser considerada ao usar um plugin.

Acessibilidade

O suporte ao teclado vem com gatilhos padrões que respondem a tecla de espaço e ao enter. Também pode ser usado para navegar entre painéis pelo uso da tecla para a ESQUERDA, e para a DIREITA.

Suporte a escritas da direita para a esquerda

Adicionar o atributo dir="rtl" a tag html irá reverter todo o comportamento direcional da inserção de dados.

Por padrão o plugin de abas irá mostrar o primeiro painél. Para mudar o painél ativo padrão, adicione na marcarção o atributo aria-selected="true" ao alvo ("target") do elemento <a/>. 

<div class="tabs" data-tabs>
    <ul>
        <li><a href="#">tab1</a></li>
        <li><a href="#" aria-selected="true">tab2</a></li>
        <li><a href="#">tab3</a></li>
    </ul>

Methods

The tabs plugin exposes the following method signatures.

.tabs(index)
Initialises the plugin with an optional index parameter. A zero based indicator for which tab to show.

Events

The tabs plugin exposes the following events allowing the developer to tap into its behaviour.

show.r.tabs
This event is fired immediately when the plugins show instance method is invoked.
shown.r.tabs
This event is fired when the plugins shown instance method is invoked once the tab has completed the animation and the new pane is visible.

Binding to these events is as follows:

// Bind to the show event.
$("YOUR_SELECTOR").on("show.r.tabs", function(event) {
    // Your custom behaviour...
});
                                               
// Bind to the shown event.
$("YOUR_SELECTOR").on("shown.r.tabs", function(event) {
    // Your custom behaviour...
});

Data API

The tabs plugin's default behaviour is bound using a markup API that allows running the plugin without writing a single line of JavaScript. If you need to disable this behaviour the following code will allow you to do so. 

// Disable the default loading behaviour
$(document).off(".r.tabs.data-api");
