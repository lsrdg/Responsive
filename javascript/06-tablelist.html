Tabela de lista

Um plugin em jQuery que povoa uma tabela com os atributos de dados necessários para mostrar uma lista em janelas de exibição menores.

Vem com uma API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Essa é a API de marcação de primeira classe do Responsive e deveria ser a primeira a ser considerada ao usar um plugin.

Métodos

O plugin de tabela de lista (tablelist plugin) expõe os seguintes métodos de assinaturas.

.tablelist()
Inicializa o plugin adicionando os atributos de dados necessários à tabela.

Eventos

O plugin Tablelist apresenta os seguintes eventos que permitem ao desenvolvedor manusear o seu comportamento.

add.r.table
Este evento é inicializado imediatamente após a instância add de método do plugin ser invocada.
added.r.table
Este evento é inicializado quando a instância added de método do plugin é invocada uma vez que o alvo tenha todos os seus atributos adicionados.

A conexão a esses eventos é feita da seguinte maneira:

// Conectar ao evento 'add'
$("O_SEU_SELETOR").on("add.r.tablelist", function(event) { 
    // O seu comportamento customizado...
});

// Conectar ao evento 'added'.
$("O_SEU_SELETOR").on("added.r.tablelist", function(event) {
    // O seu comportamento customizado...
});

API de dados

O comportamento padrão do plugin Tablelist é limitado pelo uso da API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. O seguinte código irá lhe auxiliar caso você precise desativar esse comportamento padrão. 

// Desativar a inicialização do comportamento padrão.
$(document).off(".r.tablelist.data-api");
