Carrossel

Um plugin que cria um plugin com jQuery que cria um carrossel para circular através de elementos. Usa transições de CSS para executar a animação.

O carrossel vem em dois modos: com deslizamento ("slides") ou com desvanecimento ("fade"), ambos animados por meio de transições de CSS3. Em navegadores que não suportam transições, eles irão simplesmente mudar sem nenhum efeito especial.

É possível rodar os plugins sem escrever uma linha sequer de JavaScript, pois todos os plugins vêm com uma marcação para a interface de programação de aplicação (API) voltada para a atribuição de dados. Essa é API do Responsive de primeira classe, e deveria ser a primeira a ser considerada quando usando os plugins. Comes with a markup API that allows running the plugin without writing a single line of JavaScript. This is Responsive's first class API and should be your first consideration when using a plugin.

Acessibilidade

O carrossel, por padrão, não fazer transições automaticamente. Isso faz com que leitores de tela tenham menos problemas ao acompanhar a página.

Em dispositivos que suportam toque, o carrossel irá iterar através da coleção de itens arrastando para a esquerda ou para a direita. Também ser utilizado o suporte ao teclado para navegar, quando o carrossel estiver em foco, usando as teclas para a ESQUERDA e para a DIREITA.

Suporte a escritas da direita para a esquerda

Adicionando o atributo dir="rtl" a tag html irá inverter a ordem dos itens do carrossel e todos os outros comportamentos de inserção serão invertidos.

Métodos

O Carousel expões as seguintes assinaturas de métodos ("method signatures").

.carousel(opções)
Inicia o plugin com um object de opção opcional.
Nome Tipo Descrição Valor padrão
mode cadeia de caracteres ("string") A cadeia de caracteres containing either the values fade or slide. slide
interval número O tempo em milésimos de segundos entre as transições do carrossel. 0 (Melhor para a acessibilidade)
pause cadeia de caracteres O evento ao qual limitar o comportamento de pausa hover
wrap booleano Se os itens do carrossel devem ser enrolados ou não true
keyboard booleano Se o carrossel responde ou não a eventos de teclado true
touch booleano Se o carrossel responde ou não a eventos de toque true
lazyLoadImages booleano Se alguma imagem deverá ser carregada com atraso ou não no carregamento da página true
lazyOnDemand booleano Se imagens deverão ser carregadas com atraso ao deslizar. Sobreescreve lazyLoadImages true
nextTrigger cadeia de caracteres Um seletor de jQuery que define qualquer elemento externo para para ativar o metódo next(). null
nextHint cadeia de caracteres A hidden html value that is added to the next trigger for accessibility purposes. The default language direction indicator is reversed by setting the dir="rtl" attribute on the html element. Next {Left|Right} Arrow
previousTrigger cadeia de caracteres A jQuery selector to define any external element to trigger the prev() method. null
previousHint cadeia de caracteres A hidden html value that is added to the previous trigger for accessibility purposes. The default language direction indicator is reversed by setting the dir="rtl" attribute on the html element. Previous {Left|Right} Arrow
indicators cadeia de caracteres Um seletor jQuery para definir qualquer indicador externo de <ol/> que realce a posição atual do carrossel null
.carousel("cycle")
Circula pelos itens da instância do carrossel em um ordem lógica.
.carousel("pause")
Ativa o evento de pausa da instância do carrossel fazendo com que a circulação pare.
.carousel("prev")
Faz com que a instância do carrossel circule ao item anterior.
.carousel("next")
Faz com que a instância do carrossel circule para o próximo item.
.carousel(number)
Faz com que o carrossel circule até um item em particular com um índica fornecido (baseado em zero).

Eventos

O carrossel apresenta os seguintes eventos, que permitem ao desenvolvedor manusear o seu comportamento.

slide.r.carousel
Esse evento é ativado imediatamente quando a instância do método de deslizamento (slide) do plugin é invocada. O evento de dado contém duas propriedades personalizadas:
relatedTarget
O item do carrossel que será o próximo na ordem lógica.
direction
A direção na qual o carrossel está transicionando.
slid.r.carousel
Esse evento é ativado quando a instância do método slid do plugin é invocada quando o carrossel tiver completado o seu deslizamento. O evendo de dado contém duas propriedades personalizadas:
relatedTarget
O item do carrossel que será o próximo na ordem lógica.
direction
A direção na qual o carrossel está transicionando.

A conexão a essses eventos é feita da seguinte maneira:

// Conectar ao evento "slide".
$("O_SEU_SELETOR").on("slide.r.carousel", function(event) {
    // O seu comportamento personalizado...
});
                                               
// Conectar ao evento "slid".
$("O_SEU_SELETOR").on("slid.r.carousel", function(event) {
    // O seu comportamento personalizado...
});

API de dados

O comportamento padrão do carrossel é limitado pelo uso da API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Entretanto, caso seja necessário, o seguinte código irá lhe permitir desativar esse comportamento padrão. 

// Desativar o comportamento de carregamento padrão
$(document).off(".r.carousel.data-api");
