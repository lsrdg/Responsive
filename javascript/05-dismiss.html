Dismiss - desvanecer

Um plugin em jQuery que adiciona a funcionalidade de esconder o elemento alvo dentro do elemento que o antecede. Faz uso de transições de CSS para executar a animação.

Vêm junto com a API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Essa é a API de primeira classe do Responsive e deveria ser a primeira a ser considerada ao precisar usar um plugin.

Exemplos

Adicionar o plugin dentro de qualquer elemento irá esconder o elemento anterior ao plugin combinando com o seletor fornecido no clique.

O conceito que lançou o redesenho de milhares de páginas. Por anos, web designers tem tem cobiçado precisão em layouts, lamentando os contextos variados em que os usuários fazem uso da rede, o que acaba comprometendo o desenho deles. O desenho adaptável advoca uma mudança no nosso pensamento ao desenhar, transformando essas barreiras em uma virtude. Usando grelhas fluídas, imagens flexíveis, e consultas de mídeia ("media queries") para abraçar a declinação e a fluidez da rede.

Marcação

/* Adicione dentro de qualquer elemento. */
<button role="button" data-dismiss-target="O_SEU_SELETOR_ANTECESSOR">x</button>

Métodos

O plugin de desvanecimento (Dismiss) expõe os seguintes métodos de assinaturas.

.dismiss(opções)
Inicializa o plugin com uma opção de object opcional.
options
Contém parâmetros para serem passados aos plugins.
target
Um seletor de jQuery que indica qual alvo deve ser escondido.
closeHint
Um valor de html escondido que é adicionado ao gatilho de fechamento por razões de acessibilidade. Por padrão refere a Click to close ("Clique para fechar").

Eventos

O plugin Dismiss apresenta os seguintes eventos para permitir que o desenvolvedor manusei o seu comportamento.

dismiss.r.dismiss
Este evento é ativado automaticamente assim que o método da instância dismiss do plugin Dismis é invocado.
dismissed.r.dismiss
Este evento é ativado quando a instância do método dismissed do plugin Dismiss é invocada após o alvo ter completado a animação e estar escondido.

A conexão com esses eventos é feita da seguinte forma:

// Conectar ao evento de fechamento.
$("O_SEU_SELETOR_DE_GATILHO").on("dismiss.r.dismiss", function(event) {
    // O seu comportamento customizado...
});
                                               
// Conectar ao evento fechado.
$("O_SEU_SELETOR_DE_GATILHO").on("dismissed.r.dismiss", function(event) {
    // O seu comportamento customizado...
});

API de dados

O comportamento padrão do plugin Dismiss é limitado pelo uso da API de Marcação, que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. O código abaixo irá lhe auxiliar caso você precise sobreescrever o comportamento padrão. 

// Desativar o comportamento de carregamento padrão
$(document).off(".r.dismiss.data-api");
