Navegação

Um plugin jQuery que adiciona uma navegação fora da tela à página. Faz uso de transições de CSS para executar a animação.

Vem com uma API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Essa é a API de primeira linha do Responsive e deveria ser a primeira a ser considerada ao fazer uso de algum plugin.

Se nenhum botão estiver presente como o primeiro descendente da navegação, o plugin irá automaticamente adicionar um como um gatilho.
Para renderizar do lado oposto da página, adiciona a classe .pull à navegação.

Abrir a navegação tranca a página para prevenir que haja barras de rolagem duplicadas em certas janelas de exibição.

Acessibilidade

Na inicialização um clone dos marcadores da navegação são colocados, escondidos no fim da página, tudo para assegurar que os marcadores ("landmarks") da lista de diálogo dos marcadores seja registrada e acessível. Os papéis apropriados e os atributos Aria também são adicionados ao botão e aos elementos de navegação.

Quando aberto pressionando tab ou se for focar em elementos apenas dentro do menu, shift + tab irá permitir que você saia para o topo. Pressionar esc irá fechar o menu.

Suporte à escritas da direita para a esquerda

Adicionar o atributo dir="rtl" a tag html irá inverter a posição da navegação.

Métodos

O plugin Navigation apresenta os seguintes métodos de assinatura.

.navigation()
Inicializa o plugin.
.navigation("show")
Mostra a navegação.
.navigation("hide")
Esconde a navegação.

Eventos

O plugin Navigation apresenta os seguintes eventos para permitir que o desenvolvedor altere o seu comportamento.

show.r.navigation
Este evento é ativado imediatamente quando a instância de método show do plugin é invocada.
shown.r.navigation
Este evento é ativado imediatamente quando a instância de método shown do plugin é invocada, o uma vez que o alvo tenha completado a animação e tenha sido mostrado.
hide.r.navigation
Este evento é ativado imediatamente quando a instância de método hide plugin tenha sido invocada.
hidden.r.navigation
Este evento é ativado imediatamente quando a instância de método hidden tenha sido invocada, uma vez que o alvo tenha completado a animação e esteja escondido.

A conexão a esses eventos é feita da seguinte maneira:

// Conectar ao evento 'show'.
$("O_SEU_SELETOR").on("show.r.navigation", function(event) {
    // O seu comportamento personalizado...
});
                                               
// Conectar ao evento 'shown'.
$("O_SEU_SELETOR").on("shown.r.navigation", function(event) {
    // O seu comportamento personalizado...
});
// Conectar ao evento 'hide'.
$("O_SEU_SELETOR").on("hide.r.navigation", function(event) {
    // O seu comportamento personalizado...
});
// Conectar ao evento 'hidden'.
$("O_SEU_SELETOR").on("hidden.r.navigation", function(event) {
    // O seu comportamento personalizado...
});

API de dados

O comportamento padrão do plugin Navigation é limitado pelo uso de uma API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. O seguinte código irá lhe permitir sobreescrever esse comportamento caso seja necessário. 

// Desativar o carregamento do comportamento padrão
$(document).off(".r.navigation.data-api");
