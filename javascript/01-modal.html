Modal

Um plugin em jQuery que adiciona funcionalidade modal a qualquer elemento que marcar. Faz uso de transições de CSS para criar animações.

Vem com uma API de marcação que Comes with a markup API that allows running the plugin without writing a single line of JavaScript. This is Responsive's first class API and should be your first consideration when using a plugin.

Acessibilidade

A janela moda (modal popup) pode ser fechada clicando no indicador, na janela sobreposta, ou apertando a tecla Esc.

Em dispositivos com suporte ao toque, imagens agrupadas irão iterar através de suas coleções ao serem arrastadas para a esquerda ou direita. O suporte ao teclado também pode ser utilizado para navegar por todos os alvos pelo uso das teclas para a ESQUERDA, e para a DIREITA.

Suporte a escritas da direita para a esquerda

Adicionar o atributo dir="rtl" a tag html irá reverter todos os comandos direcionais feito pela usuária.

Métodos

O modal expõe os seguintes métodos de assinatura.

.modal(options)
Inicializa o plugin com opções de objeto (object) opcionais.
Nome Tipo Descrição Valor padrão
target string O alvo a ser mostrado no modal. Tanto um endereço de url relativo ou absoluto quanto um seletor de jQuery. null
modal string Um seletor jQuery que atribui um gatilho para substituir todos os outros gatilhos de fechamento. false
external boolean Se o alvo está em uma página diferente. Usado para forçar o modal a reconhecer endereços de url externos quando a detecção automática falhar. false
group string Um seletor de jQuery que contém um identificador de grupo para a criação de galerias. null
image boolean Se o alvo é uma imagem. Usado para forçar o comportamento de imagem caso a detecção automática falhar. false
immediate boolean Se for o caso de mostra o modal imediatamente. false
iframe boolean Se for o caso de usar um iframe. Usado para forçar o comportamento de iframe caso a detecção automática falhar. false
iframeScroll boolean Se o iframe deve ou não ter barras de rolagem. true
fitViewport boolean Se a largura do conteúdo carregado deve ou não ser forçada a preencher a janela de exibição. true
keyboard boolean Se o controle por teclado deve ou não ser ativado. true
touch boolean Se a Lightbox deve ou não responder a eventos de toque. true
title string O html para ser inserido como o título do modal. null
description string O html para ser inserido como a descrição do modal no rodapé. null
next string O html a ser inserido no indicador do próximo item. >
nextHint string Um valor html escondido que é adicionado - por razões de acessibilidade - ao gatilho para o próximo item. O indicador de direção da língua padrão é revertido configurando o atributo dir="rtl" no elemento html. Next {Left|Right} Arrow
previous string O html a ser inserido no indicador do item anterior. <
previousHint string Um valor html escondido que é adicionado - por razões de acessibilidade - ao gatilho para o próximo item. O indicador de direção da língua padrão é revertido configurando o atributo dir="rtl" no elemento html. Previous {Left|Right} Arrow
closeHint string Um valor html escondido que adicionado por razões de acessibilidade e tem a função de fechar o modal. Close (Esc)
errorHint string O html contendo uma mensagem que será inserida no modal quando um erro de ajax ocorrer.. <p>Aconteceu um erro.</p>
mobileTarget string Um endereço url alternativo para visualização a partir de dispositivos móveis. null
mobileViewportWidth number A largura máxima da janela de exibição para apresentar o url alternativo. Isso pode ser fornecido tanto passando um dos alcances da grelha xxs, xs ,s,m, ou l ou por um valor em pixel. xs
.modal("show")
Mostra o modal.
.modal("hide")
Esconde o modal.
.modal("prev")
Circula pela instância do modal para o item anterior em um grupo.
.modal("next")
Circula pela instância do modal para o próximo item em um grupo.

Eventos

O Modal apresenta os seguintes eventos para permitir que o desenvolvedor manusei o seu comportamento.

show.r.modal
Esse evento é ativado imediatamente quando a instância de método show do plugin é invocada.
shown.r.modal
Esse evento é ativado quando a instância de método shown do plugin é invocada uma vez que o alvo tenha completado a animação e tenha sido mostrado ("shown").
hide.r.modal
Esse evento é ativado imediatamente quando a instância de método hide do plugin é invocada.
hidden.r.modal
Esse evento é ativado quando a instância de método hidden do plugin é invocada e uma vez que o alvo tenha completado a animação e estiver escondido.
error.r.modal
Esse evento é ativado quando o plugin encontra um erro ao tentar carregar conteúdo em ajax.

A conexão à esses eventos é feita da seguinte maneira:

// Conectar ao evento show.
$("O_SEU_SELETOR_DE_GATILHO").on("show.r.modal", function(event) {
    // O seu comportamento personalizado...
});
                                               
// Conectar ao evento shown.
$("O_SEU_SELETOR_DE_GATILHO").on("shown.r.modal", function(event) {
    // O seu comportamento personalizado...
});
// Conectar ao evento hide.
$("O_SEU_SELETOR_DE_GATILHO").on("hide.r.modal", function(event) {
    // O seu comportamento personalizado...
});
// Conectar ao evento hidden.
$("O_SEU_SELETOR_DE_GATILHO").on("hidden.r.modal", function(event) {
    // O seu comportamento personalizado...
});
// Bind to the ajax error event.
$("O_SEU_SELETOR_DE_GATILHO").on("error.r.modal", function(event) {
    // O seu comportamento personalizado...
});

API de dados

O comportamento padrão do plugin modal é limitado pelo uso da API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Caso você precise desativar esse comportamento, o seguinte código irá lhe auxiliar. 

// Desativar o comportamento de carregamento padrão
$(document).off(".r.modal.data-api");
