Autosize - Redimensionamento automático

Um plugin jQuery que possibilita o redimensionamento vertical automático de áreas de texto. Útil para assegurar que áreas de texto não desorganizem a página. Faz uso de transições de CSS para a animação.

Vem com uma API de marcação que permite rodar o plugin sem precisar escrever uma linha sequer de JavaScript. Essa é a API de primeira classe do Responsive e deveria ser a primeira a ser considerada para utilizar algum plugin.

Atenção!

O plugin Autosize estende a função val() do jQuery para ativar um evento: input. Isso permite que o desenvolvedor conecte e manusei o evento quando configurar o conteúdo programaticamente.

Exemplos

Atenção!

Caso você rode o plugin sem usar a API de marcação, adicione a classe .autosize ao plugin para conseguir o estilo de comportamento desejado.

Marcação

<!-- Marcação padrão -->
<textarea id="textarea1" name="textarea1" data-autosize></textarea>

Métodos

O autosize apresenta os seguintes métodos de assinaturas.

.autosize()
Inicializa o plugin.
.autosize("size")
Ativa o plugin de modo que o redimensionamento combine com o texto.

Eventos

O plugin autosize apresenta os seguintes métodos, permitindo que o desenvolvedor manusei o comportamento.

size.r.autosize
Esse evento é ativado automaticamente quando o método da instância size é invocado.
sized.r.autosize
Esse evento é ativado automaticamente quando o método da instância sized é invocado uma vez que a área de texto tenha completado a animação e tenha sido redimensionada.

A conexão a esses eeventos é feita da seguinte forma:

// Conectar ao evento "size".
$("SEU_SELETOR").on("size.r.autosize", function(evento) {
    // O comportamento customizado por você...
});
                                               
// Conectar ao evento "sized".
$("YOUR_SELECTOR").on("sized.r.autosize", function(evento) {
    // O comportamento customizado por você...
});

API de dados

O comportamento padrão do Autosize é delimitado pelo uso da API de Marcação que permite rodas o plugin sem precisar escrever uma linha sequer de JavaScript. Caso for preciso, o código abaixo irá lhe permitir desativar esse comportamento. 

// Desativar o comportamento de carregamento padrão
$(document).off(".r.autosize.data-api");
