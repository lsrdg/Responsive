Baixe o Responsive

O jeito mais simples e rápido de conseguir a Responsive baixar as versões pre-compiladas e minificadas dos nossos CSS, e JavaScript. Nenhuma documentação ou arquivos do código fonte incluídos.

Download Responsive

Você também pode instalar o Responsive com o Bower:

bower install responsive

Se você está se sentindo mais aventurosa e quer mexer no Responsive antes de usar, você pode baixar o código completo de Sass e JavaScript direto do Github. Desse jeito, vai precisar usar ogulp.js para construir.

Montando o seu HTML

Dentro do arquivo em zip você encontrará também um arquivo HTML contendo o modelo padrão para incluir o Responsive em sua página.
Monte o seu modelo como o exemplo abaixo. 

<!DOCTYPE HTML>
<html>
    <head>
        <meta charset="utf-8" />
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1" />
        <meta name="description" content="Your Page Description." />
        <title>O Título da sua página</title>
 
        <link href="responsive.min.css" rel="stylesheet" />
    </head>
    <body>
        O Conteúdo Do Seu Site
    
        <!-- Scripts at the bottom for improved performance. -->
        <!-- jQuery, required for all responsive plugins. -->
        <script src="vendor/jquery-2.1.4.min.js"></script>
        <script src="responsive.min.js"></script>
    </body>
</html>

Suporte ao navegador

Responsive é construído para funcionar melhor em navegadores mais recentes, tanto em computadores quanto em dispositivos móveis. Tudo foi construído com melhoramento progressivo em mente, o que significa que navegadores mais antigos podem mostrar componentes estilizados de forma diferente, mas ainda assim totalmente funcionais.

Navegadores suportados

Especificamente, as versões mais recentes dos seguintes navegadores são suportadas:

  • Chrome (Mac, Windows, iOS, e Android)
  • Internet Explorer (Windows - mais recente e IE9, IE10, WP7/8)
  • Firefox (Mac, Windows)
  • Opera (Mac, Windows)
  • Safari (Mac e iOS apenas, já que a versão pro Windows está sendo abandonada)

Safari

Safari antes da v7.1 para OS X e Safari para iOs v8.0 não calculam percentagens de forma apropriada e vão renderizar 12 colunas individuais definidas usando as classes col-{range}-1 levemente mais pontudas do que deveriam. Veja #40.

Infelizmente, além de atualizar o seu navegador não há muito que possa ser feito sem recorrer a remendos para dar um jeito nas percentagens usando JavaScript ou usando a classe push-{range} ou push-{range}-only para compensar a última coluna.

Internet Explorer 9

Internet Explorer 9 não tem suporte para as transições de CSS e por isso todos os plugins que fazem uso de transições irão operar sem animações

Acessibilidade

Responsive é construído usando padrões comuns da rede e tem sido desenhado para suportar Tecnologia Assistiva. Todas as cores foram escolhidas conforme a proporção recomendade de 4.5:1 para assegurar que usuários com baixa capacidade de visão ou com daltonismo sejam atendidos apropriadamente.

Todos os plugins do Responsive foram construídos desde a base para garantir a acessibilidade ao mouse, toque e teclado. Eles também adicionam a atribuição ARIA onde for necessário para assegurar que leitores de tela (screenreaders) possam seguir a funcionalidade de cada plugin.

Línguas da direita para esquerda

Responsive tem suporte nativo a línguas da direita-para-a-esquerda como o hebreu ou árabe. Todos os aspectos da estrutura, desde a coluna de grades até o comportamento de deslocamento direcional, possuem contrapartes da 'direita-para-a-esquerda'.

Para ativar o suporte a línguas da direira-para-a-esquerda adicione o atributo dir="rtl" para a tag html.
