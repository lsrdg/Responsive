Responsive CSS

Responsive difere de outras estruturas de CSS (frameworks) no fato de que é construído especialmente para não forçar um desenho já estabelecido encima do desenvolvedor. Pelo contrário, Responsive foi empacotado com funcionalidades úteis para permitir que você construa uma verdadeira experiência de desenho responsável, independente do navegador.

Um poderoso sistema de grade, onde está incluído o suporte a línguas escritas da direita para a esquerda por padrão. Formulários, butões, e outros elementos também foram normalizados e incrementados para eliminar bugs (defeitos) e fazer ficar mais fácil a construção da sua página.
