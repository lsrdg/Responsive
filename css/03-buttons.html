Elementos para botões

O estilo dos botões foi desenhado com o mínimo necessário de CSS, como tudo mais no Responsive. Os estilos foram normalizados independentemente dos estados dos botões, com nenhuma adição de classes extras.

Atenção!

De forma proposital, os elementos com [role="button"] não foram estilizados para combinar com as suas contrapartes nativas. Isso foi feito por razões de acessibilidade, já que acreditamos que isso iria promover mal-uso.

Estilos de botão padrão

Todos os estilos são baseados nos atributos, cobrindo todos os quatro estados dos botões:

  • :hover
  • :focus
  • :active
  • :disabled

Botão

Elemento de botão padrão: button.

Botão : de inserção

Elemento padrão de botão para inserção: input[type="button"].

Botão : para enviar

Elemento padrão para o envio de dados input[type="submit"].

Botão : para reiniciar

Elemento de botão padrão para reiniciar: input[type="reset"].

Atenção!

Recomendamos fortemente o uso do elemento <button></button> ao invés de usar o elemento <input /> para estilizar toda a marcação de butões.

Estado desativado

Hyperlinks em estado 'desativado' são estilizados adicionando a classe .disabled. O desenvolvedor teria que adicionalmente usar JavaScript para prevenir o comportamento padrão.

Botões na linha

Para criar botões que aparecem diretamento na linha e com o campo para inserção de dado, posicione o elemento dentro de um container (div) com a classe .input-group e o botão em um vão (span) com a classe .input-group-addon.
