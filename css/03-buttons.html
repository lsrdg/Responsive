Elementos para botões

O estilo dos botões foi desenhado com o mínimo necessário de CSS, como tudo mais no Responsive. Os estilos foram normalizados independentemente dos estados dos botões, com nenhuma adição de classes extras.

Atenção!

We have deliberately not styled elements with [role="button"] to match their native counterparts. This is for accessibility reasons as we believe that is will promote misuse to do so.

Default Button Styles

All styles are attribute based with all four button states covered:

  • :hover
  • :focus
  • :active
  • :disabled

Heads Up!

We strongly recommend using the <button></button> element over the <input /> element for all button style markup.

Disabled State

Hyperlink disabled state are styled by adding the class .disabled. The developer would have to additionally use JavaScript to prevent the default behaviour.

Inline Buttons
