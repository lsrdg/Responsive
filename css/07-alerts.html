Alertas

Caixas de alerta são usadas para trazer informação para a perspectiva do usuário, Responsive vem com um padrão que provê um desenho flexível independente do tamanho da janela de exibição e que pode ser extendido.

A funcionalidade para fechar a caixa é provida pela JavaScript Data API.
