Blocos de código

Elementos de código foram normalizados e estilizados com apenas o mínimo possível de CSS, tudo para permitir um comportamento consistente e fácil substituição.

Código em linha

Coloque o seu código em linha em um bloco de <code></code> para realçar um fragmento de código de computador.

Use o elemento samp. Você pode usar <samp></samp> para apresentar amostr apresentar amostra de saída de um computador ou sistema operacional. Entrada de teclado ou comandos de voz podem ser mostrados usando o elemento <kbd></kbd>, por exemplo: press shift + f5

Blocos pré-formatados

Coloque o seu bloco de amostra em um bloco de <pre></pre>.

Esse é o construtor Panel:

function Panel(element, canClose, closeHandler) {
  this.element = element;
  this.canClose = canClose;
  this.closeHandler = function () { 
    
    if (closeHandler) {
    
        closeHandler(); 
    }
  };
}

Altura fixada

Adicione a classe .pre-scrollable para limitar blocos pré configurados a uma altura fixada.

pre, code, samp, kbd {
    background-color: #f7f7f9;
    border: 1px solid #e1e1e8;
    color: #dd1144;
    padding: .5em;
}
code {
    padding: .067em .25em;
    white-space: nowrap;
}
pre code,
pre samp {
    border: none;
    padding: 0;
    white-space: pre-wrap;
}
.pre-scrollable {
    max-height: 20em;
    overflow-y: scroll;
}
