Mídia

Elementos de mídia que escalam junto com o seu navegador. Imagens, Iframes e Vídeos foram todos estilizados para assegurar que se adaptem a sua janela de exibição.

O código aqui foi baseado na pesquisa e trabalho de Theirry Koblentz, Anders Andersen, Niklaus Gerber, e Jeff Hobbs.

Imagens

As imagens são todas adaptáveis por padrão, assegurando que elas não transbordem para além dos limites de seus respectivos containers.

Images restringidas

Restrigidas vertical e horizontalmente e centralizadas. Coloque a imagem em um container com a classe .media.

Adicionar a classe .four-by-three irá mudar a razão do aspecto do padrão de 16:9.

Atenção!

As versões 9/10/11 do Internet Explorer não escalam imagens svg corretamente quando carregadas usando a tag <img/>.

De modo a prover uma adaptabilidade básica, adicionamos algumas classes de CSS que replicam, em grande parte, outros comportamentos do navegador. Entretanto, isso irá forçat a svg a preencher a largura total do container que à contém, portanto se você requer que uma imagem svg não faça isso, você terá que sobreescrever a svg específica com CSS para aplicar uma largura máxima definida.

Marcação

<div class="media">
    <iframe src="http://www.youtube.com/embed/eRW84aVCMWo" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen></iframe>
</div>
