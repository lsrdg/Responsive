Normalize

Responsive fornece vários blocos de de construção que são especificamente desenhados para acelerar o seu desenvolvimento, padronizando componentes comuns de estilos normalizados.

O CSS básico para isso, foi cuidadosamente adaptado do Normalize.css; um projeto de código aberto projetado por Nicolas Gallagher e Jonathan Neal.

Nas palavras deles mesmos:

Normalize.css é um pequeno arquivo de CSS que provem uma melhor consistência através dos navegadores no estilo padrão de elementos de HTML. É uma alternativa moderna, pronta para HTML5, ao CSS tradicional já estabelecido.

Tipografia

Responsive não adiciona nenhum estilo adicional aos elementos básicos da tipografia como <p/>. Marcas de caçalhos e outros elementos estão conforme os padrões normalizados, para suportar múltiplos navegadores.

Auxiliadores extras de visibilidade

São classes extras que te auxiliam a realizar tarefas adicionais de visibilidade.

.hidden
Esconde um elemento tanto de leitores de tela (screen-readers) quanto de navegadores.
.visuallyhidden
Esconde um elemento visualmente, mas deixa disponível para leitores de tela.
.visuallyhidden.focusable
Estende a classe .visuallyhidden para permitir que o elemento seja 'focável' quando navegado pelo teclado.
.invisible
Esconde um elemento tanto visualmente quanto de leitores de tela, mas mantém o layout.
.ir
Esconde o texto de um elemento, de modo a permitir que uma imagem seja usada (ao invés do texto); para funcionar, requer que a altura (height) do elemento seja estabelecida pelo CSS. Útil para logos.

Auxiliares de layout

Responsive também vem empacotado com classes úteis para te ajudar a distribuir os elementos dentro de cada coluna da grade.

Classes flutuantes

Empurre ou puxe os elementos, fazendo com que flutuem para as extremidades dos elementos que os cercam. Adicione a classe .clearfix ao elemento que os elementos flutuantes, para que seja possível conter os elementos flutantes sem precisar fazer uso de marcação de apresentação.

.push

.pull

Atenção!

Tanto .push quanto .pull possuem .push/.pull-{grid range} e .push/.pull-{grid range}-only como variantes para o alcance de cada grade.

Classes de exibição

As seguintes classes estão disponíveis para alterar o layout (traço) de elementos

.block
Muda o estilo de exibição de um elemento para block.
.inline-block
Muda o estilo de exibição de um elemento para inline-block.
.inline
Muda o estilo de exibição de um elemento para inline.

Atenção!

.block, .inline-block, e .inline possuem {display}-{grid range} e {display}-{grid range}-only como variantes para o alcance de cada grade.

Auxiliares de layout de texto

Responsive provem métodos alinhar um texto dentro de um container

As seguintes classes estão disponíveis para alterar o layout (traçado/desenho) de texto

.text-push
Alinha o texto para direita do container. Esquerda no modo de língua da-esquerda para a direita.
.text-pull
Alinha o texto para esquerda do container. Direita no modo de língua da-esquerda para a direita.
.text-center
Alinha o texto para o centro do container.

Atenção!

.text-pull, .text-push, e .text-center possuem {text}-{grid range}-{align}
e {text}-{grid range}-{align}-only como variantes para o alcance de cada grade.
