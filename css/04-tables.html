Elementos de tabela

Responsive provêm tabelas com tres estilos básicos diferentes, com o mínimo de CSS. Fácil para encaixar onde for preciso e editar de acordo com as suas necessidades.

Layout básico

A tabela básica não possui nenhuma classe adicionada a si. Nós apenas usamos os padrões, como definidos no Normalize.css, e adicionamos algumas propriedades para que se encaixe melhor com desenhos fluídos.

Tabelas adaptáveis

Há dois tipos a mais de tabelas adaptáveis na biblioteca. Um tipo simples que adiciona uma barra horizontal a sua tabela em janelas de exibição menores, e um outro tipo mais flexível, que cresce a partir de uma lista em dispositivos menores.

Tabela que embrulha o excesso

Coloque a tabela em um container com a classe .table-scrollable.

Tabela de lista

Tabela de lista faz uso de atributos de dados de html5 para renderizar tabelas verticalmente em janelas de exibição menores.
Adicionar o atributo data-table-list irá automatizar isso para você, fazendo uso da JavaScript Dado API.
