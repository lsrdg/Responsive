Elementos de listas

Listas adaptáveis com preenchimento e margem que vão escalando com o tamanho base da fonte. A estilização de CSS é minimalista e existe apenas para normalizar o comportamento e prover encaixes de estilos para desenvolvimento futuros.

Suporte a línguas com escrita da direita para a esquerda

Adicionar o atributo dir="rtl" a tag html irá reverter o layout das listas para permitir línguas escritas da direita para a esquerda.
